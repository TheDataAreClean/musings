Notes can contain **bold**, *italic*, and `inline code`. They can link to [other pages](/ideas/) or [external sites](https://www.example.com). They can combine: ***bold italic*** works fine. So does ~~strikethrough~~, though it reads as a correction rather than style.

This note tests that inline formatting renders correctly in a short-form context where there are no headings, no sections — just prose.
